# 线程池创建方法

| 方法名                    | 介绍                                   |
| ------------------------- | -------------------------------------- |
| newSingleThreadExecutor() | 创建只有一个线程的线程池               |
| newFixedThreadPool(int n) | 创建固定大小的线程池                   |
| newCachedThreadPool()     | 创建一个不限线程数量的线程池           |
| newScheduledThreadPool()  | 创建一个可定期或者延时执行任务的线程池 |
